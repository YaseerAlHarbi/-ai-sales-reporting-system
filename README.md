## 🎯 What It Does
Fully automated sales reporting pipeline that replaces manual reporting with AI-generated insights.

- Processes 2,000+ sales transactions across 4 global regions & 4 categories
- Auto-generates a styled Excel report with 6 analytical sheets + charts
- Claude AI writes narrative analysis automatically (natural language)
- Interactive web dashboard with 6 analytical views & real-time KPIs
- Anomaly detection flags unusual patterns automatically

## ⚡ Results
| Metric | Value |
|--------|-------|
| Manual Reporting Time Saved | ~90% |
| Records Processed | 2,000 orders |
| Pipeline Runtime | < 10 seconds |
| Output Files | Excel + HTML Dashboard + AI Report |

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| Python · Pandas · NumPy | Data processing & pipeline |
| OpenPyXL | Excel report generation |
| Chart.js | Interactive dashboard |
| Claude AI API | Natural language insights |

## 🚀 Run It
```bash
pip install -r requirements.txt
python main.py
```

## 📁 Outputs
- `Dashboard.html` → Open in browser for interactive charts
- `Sales_Report_2024.xlsx` → Full Excel report with 6 sheets
- `AI_Narrative_Report.txt` → AI-written business analysis
